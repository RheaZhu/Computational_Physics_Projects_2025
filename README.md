# Computational_Physics_Projects_2025
Repository of computational physics projects completed in 2025

This repository contains three projects: 

1) Modelling Dual Deployment Rocket Recovery Systems Under Constant Wind Conditions
2) Classical Electron Orbital Trajectories Under Variable Initial Velocity Angle and Nucleic Charge
3) Simulating U-235 Fission in Water Coolant Environments Under Variable Boric Acid Concentrations & Lowly-Enriched Uranium Number Densities
